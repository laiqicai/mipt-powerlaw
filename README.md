# Elimination of MIPTs in Quantum Circuits with Power-Law Connectivity

**Paper:** "Elimination of Measurement-Induced Phase Transitions in Quantum Circuits with Power-Law Connectivity"  
**Author:** Qicai Lai  
**Target journal:** SciPost Physics

## Main result

Monitored random circuits with power-law gate connectivity $w(d) \sim d^{-\alpha}$ have a sharp critical boundary $\alpha_c = 1$: the volume-law phase extends to all measurement rates $p < 1$ when $\alpha < 1$. This improves the Minato et al. bound ($\alpha_c = 2$) by a factor of two and coincides with the Kac criterion of classical long-range models.

Free-fermion Hamiltonian simulations reveal that $\alpha_c = 1$ is specific to circuit dynamics — in Hamiltonian systems with Kac normalization, the scaling exponent $\beta(\alpha)$ is *reversed* due to bond-strength dilution.

## Quick start

### 1. Compile and run Clifford simulations

```bash
g++ -O3 -std=c++17 -o carroll carroll_full.cpp
./carroll --all          # Full run (~hours, generates all CSV data)
./carroll --quick        # Fast sanity check (~2 min)
```

Individual modes:
```bash
./carroll --standard     # Fig 1(a,d): nearest-neighbor brickwork
./carroll --longrange    # Fig 1(b,c): α=0 long-range
./carroll --alpha        # Fig 2, Table I: α scan
./carroll --deltaS       # Fig S1: rate equation validation
./carroll --fss          # Fig S2/S3: finite-size scaling β(L)
./carroll --marginal     # Fig S4: marginal case α=1
./carroll --therm        # Fig S6: thermalization convergence
./carroll --gates        # Fig S8: gate ensemble comparison
```

### 2. Run Haar-random check

```bash
pip install numpy scipy
python haar_check.py     # Fig S5 data → results/haar_check.csv
```

### 3. Run free-fermion Hamiltonian simulations (GPU recommended)

```bash
# Install PyTorch with CUDA (adjust cu121 to match your CUDA version)
pip install torch --index-url https://download.pytorch.org/whl/cu121

python free_fermion_mipt.py              # Full run (~5 hours on RTX 3050)
python free_fermion_mipt.py --quick      # Quick test (~30 min)
python free_fermion_mipt.py --resume     # Resume interrupted run
```

Output: `results/free_fermion_mipt_gpu.csv` → Fig S7

### 4. Generate all figures

```bash
pip install matplotlib numpy
python make_all_figures.py               # All figures
python make_all_figures.py --main        # Main text only
python make_all_figures.py --supplement  # Supplement only
DATADIR=my_results python make_all_figures.py  # Custom data directory
```

### 5. Compile paper

```bash
pdflatex paper_with_proof.tex && pdflatex paper_with_proof.tex
pdflatex supplement.tex && pdflatex supplement.tex
```
